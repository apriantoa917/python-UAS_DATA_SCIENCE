Untuk dapat menggunakan perhitungan lakukan langkah berikut ini :
1. paste 1 folder ini kedalam path jupiter notebook
2. masuk kedalam path dimana folder ini berada dan buka file dengan ekstensi ipnyb
3. jalankan file ipnyb dengan memilih toolbar kernal -> restart & run all -> konfirmasi -> tunggu hingga semua source code dijalankan

informasi :
untuk melakukan pengubahan data dapat membuka file excel yang ada (file import_data_awal ataupun deteksi_gejala).

setelah data di file excel, silahkan simpan perubahan dan jalankan ulang semua kode sesuai dengan petunjuk nomor 3.
